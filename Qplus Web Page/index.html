    Tipos de Instalación

    EN SITIO

    Instalación del software en el servidor o infraestructura del cliente. El servidor, sus licencias y los servicios de IT son recursos propios del cliente.

    LA NUBE

    Software alojado en la nube de QPLUS. Reduce el impacto financiero de su adquisición y brinda seguridad y oportunidad de la información.

    MULTIEMPRESA

    Automatiza y administra diferentes sistemas de gestión de diferentes compañías de un mismo grupo o diferentes sectores de la organización, compartiendo e independizando la información por empresa; se instala una sola licencia y las empresas usuarias acceden al servidor vía internet o intranet.

    LA NUBE

    Para el caso del servicio ofrecido en la Nube (QPLUS NUBE), el servidor de D&M que se encuentra en la nube cuenta con las características necesarias para la instalación de la aplicación y de la base de datos.

    El servicio QPLUS NUBE se rige bajo las condiciones de uso del software QPLUS NUBE como servicio que se encuentran en la página web de D&M y la política de tratamiento de datos personales de D&M Software Solutions que también se encuentra en la página web de D&M (dymsites.co).

    D&M Software Solutions entregará el producto para el número de usuarios definidos por el cliente. En el evento en que se requieran más usuarios, el usuario administrador tendrá la opción de crearlos sin ningún costo adicional, hasta el límite de usuarios de la licencia.

    Requisitos técnicos necesarios en el servidor donde se instalará la aplicación son:

    • Windows server 2012 o superior.
    • IIS 10.0 o superior
    • Microsoft Net framework 4.5 o superior
    • Asignación de memoria: 4 GB disponibles para la aplicación.
    • Espacio en disco disponible de 6 GB.
    • 2 core como mínimo.
    • Certificado SSL

    Requisitos técnicos necesarios en el servidor donde se instanciará la base de datos:

    • BD SQL 2012
    • Asignación de memoria: 2 GB disponibles para la aplicación
    • Espacio en disco disponible de 5 GB proyectado a 5 años
    • 1 core como mínimo
